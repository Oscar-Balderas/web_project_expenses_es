# TripleTen web_expense_tracker

Aplicación web interactiva para gestionar un presupuesto personal y registrar gastos por categoría.

Permite asignar un presupuesto, agregar gastos dinámicamente y visualizar estadísticas en tiempo real.

---

## 🚀 Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Manipulación del DOM
- Cálculos dinámicos

---

## 📊 Funcionalidades

- Asignar presupuesto inicial
- Agregar y eliminar gastos
- Cálculo automático de:
  - Gastos totales
  - Gasto promedio
  - Saldo restante
- Estadísticas por categoría
- Identificación de la categoría con mayor gasto
- Actualización dinámica sin recargar la página
